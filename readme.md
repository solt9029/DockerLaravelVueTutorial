# Laravel + Vue

## 環境構築

```
cd LaravelVueTutorial
cp .env.sample .env
npm install
composer install
npm run dev
cd docker.laravelvuetutorial
docker-compose build
docker-compose up -d
```

[http://docker.app:8011](url)で確認することができます

## バージョン

- node: 7.2.1

- npm: 4.5.0

- docker: 1.13.1

- docker-compose: 1.11.1

- composer: 1.3.2
